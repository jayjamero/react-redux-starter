# React Redux Starter Full
This is a boiler plate React/Redux project that contains build tools for React, Sass and TDD based workflows. 

## Installation Steps
1. `npm install`
2. `gulp serve`

## Extras
* `gulp test` - Runs mocha tests
* `gulp dist` - Create a release candidate
